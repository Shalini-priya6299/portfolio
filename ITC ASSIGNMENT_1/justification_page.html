~~Welcome to my Justification Page!

Here, i'll talk about my justification for why i created only one file for all html file. I want all my pages and its background to be uniform that's why I created only one css file. Having uniform background, uniform font, uniform font-size looks simple yet elegent. That's what i want to do here.

Here are the reasons why I created a single css file for all html files:

  • Consistency in Design : Ensures a uniform look across all subjects and the bio page, reflecting professionalism.
  • Ease of Maintenance: Simplifies updates and adjustments by centralizing styles in one file, that is easy to maintain.
  • Simplified Development Process: Streamlines setup by avoiding multiple stylesheets, minimizing potential errors.
  • Modesty with Elegent Design: Organizes styles by subject, allowing easy additions or modifications without disruption.
