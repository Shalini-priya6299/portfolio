# Portfolio / ITC Assignment

Welcome to this portfolio repository. 

## About
This repository is intended to showcase work and assignments. The provided assignment folder is a self-contained site or project that can be opened locally for review.

## How to view the project locally
1. Clone the repository:
   - git clone https://github.com/Shalini-priya6299/portfolio.git
2. Open the folder in your file explorer:
   - cd portfolio/ITC\ ASSIGNMENT_1
3. Open the main HTML file (for example, index.html) in your browser:
   - double-click index.html, or
   - serve the folder locally (recommended) to avoid CORS issues:
     - Python 3: python -m http.server 8000
     - Then open http://localhost:8000 in your browser.

If you don't see an index.html, open the relevant HTML files inside the ITC ASSIGNMENT_1 folder.
